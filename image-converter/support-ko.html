❓ 자주 묻는 질문

Image Converter는 어떤 이미지 형식을 지원하나요?
Image Converter는 다음 형식 간 변환을 지원합니다:
  • 입력 형식: JPG, JPEG, PNG, HEIC, HEIF, WebP, PDF
  • 출력 형식: JPG, PNG, WebP, PDF
새로운 형식 지원을 지속적으로 추가하고 있습니다. 특정 형식이 필요하시면 기능 요청으로 알려주세요.
변환 시 이미지 품질이 압축되나요?
출력 품질을 완전히 제어할 수 있습니다:
  • JPG 형식: 압축 품질 조절 가능(1–100%)
  • PNG 형식: 무손실 압축, 원본 품질 유지
  • WebP 형식: 손실 또는 무손실 압축 선택
  • HEIC 형식: 높은 품질을 유지하면서 파일 크기 감소
기본 설정은 품질과 파일 크기 사이의 좋은 균형을 제공합니다.
이미지가 서버에 업로드되나요?
절대 업로드되지 않습니다. Image Converter는 완전한 로컬 처리 아키텍처를 사용합니다:
  • 모든 변환은 기기에서 수행됩니다
  • 앱 사용에 인터넷 연결이 필요하지 않습니다
  • 이미지는 기기를 벗어나지 않습니다
  • 개인정보 보호 최우선 설계 철학을 따릅니다
데이터는 완전히 사용자의 통제 하에 있습니다.
일괄 변환을 지원하나요?
네! Image Converter는 강력한 일괄 변환을 지원합니다:
  • 여러 이미지를 한 번에 선택
  • 모든 이미지에 동일한 변환 설정 적용
  • Photos 또는 Files로 일괄 내보내기
  • 원본 파일 구성 유지
무료 버전에는 일괄 변환 수량 제한이 있습니다. Pro로 업그레이드하면 무제한 일괄 변환이 가능합니다.
변환된 이미지는 어디에 저장되나요?
이미지 저장 위치를 선택할 수 있습니다:
  • Photos: iOS 사진 라이브러리에 직접 저장
  • Files: iCloud Drive 또는 로컬 파일에 저장
  • 공유: 다른 앱으로 직접 공유
설정에서 기본 저장 위치를 설정하거나, 변환할 때마다 선택할 수 있습니다.
Pro 구독에는 어떤 기능이 포함되나요?
Pro 구독으로 모든 프리미엄 기능이 해금됩니다:
  • 무제한 일괄 변환
  • 모든 출력 형식
  • 고급 품질 설정
  • 사용자 지정 출력 크기
  • 광고 제거
  • 우선 기술 지원
구독은 App Store를 통해 관리되며 언제든지 취소할 수 있습니다.
구독을 취소하려면 어떻게 하나요?
다음 단계를 따라 구독을 취소할 수 있습니다:
  1. iOS 기기의 설정 앱을 여세요
  2. 상단의 이름을 탭하세요
  3. "구독"을 선택하세요
  4. Image Converter Pro를 찾으세요
  5. "구독 취소"를 탭하세요
취소 후에도 현재 구독 기간이 종료될 때까지 Pro 기능을 계속 사용할 수 있습니다.
앱이 충돌하면 어떻게 해야 하나요?
앱이 충돌하면 다음 단계를 시도해 보세요:
  1. 앱을 완전히 종료(앱 전환기에서 제거)한 후 다시 여세요
  2. 기기를 재부팅하세요
  3. 앱 업데이트를 확인하세요
  4. iOS 시스템 업데이트를 확인하세요
  5. 앱을 삭제하고 재설치하세요(참고: 저장된 이미지는 삭제되지 않습니다)
문제가 지속되면 기기 모델과 iOS 버전을 함께 지원 팀에 문의해 주세요.
어떤 iOS 버전이 지원되나요?
Image Converter는 iOS 15.0 이상을 지원합니다.

최상의 경험을 위해 다음을 권장합니다:
  • 최신 iOS 버전 사용
  • 앱을 최신 버전으로 유지
  • 기기에 충분한 저장 공간 확보
환불을 요청하려면 어떻게 하나요?
환불 요청은 Apple을 통해 처리되어야 합니다:
  1. reportaproblem.apple.com에 접속하세요
  2. Apple ID로 로그인하세요
  3. Image Converter 구독 또는 구매 내역을 찾으세요
  4. "문제 보고"를 탭하세요
  5. 환불 사유를 선택하고 제출하세요
환불 승인은 Apple의 재량에 따릅니다. 질문이 있으시면 Apple 지원에 직접 문의하실 수도 있습니다.

🔄 빠른 시작: 단일 이미지 변환

  1. Image Converter 앱을 여세요
  2. 메인 화면의 "+" 버튼 또는 "이미지 선택"을 탭하세요
  3. 사진 라이브러리에서 변환할 이미지를 선택하세요
  4. 대상 형식을 선택하세요(JPG, PNG, WebP 또는 PDF)
  5. 품질 및 크기 설정을 조정하세요(선택 사항)
  6. "변환" 버튼을 탭하세요
  7. 저장 위치를 선택하세요(Photos 또는 Files)

📦 여러 이미지 일괄 변환

  1. "일괄 변환" 옵션을 탭하세요
  2. 여러 이미지를 선택하세요(스와이프하여 다중 선택)
  3. 선택을 확인하고 "다음"을 탭하세요
  4. 통일된 출력 형식과 품질을 설정하세요
  5. "일괄 변환 시작"을 탭하세요
  6. 처리가 완료될 때까지 기다리세요
  7. 변환된 이미지를 일괄 저장하거나 공유하세요

⚙️ 사용자 지정 변환 설정

  1. 변환 화면의 설정(톱니바퀴) 아이콘을 탭하세요
  2. 출력 품질 슬라이더를 조정하세요(JPG/WebP 전용)
  3. 출력 크기를 설정하세요(원본, 사용자 지정 또는 프리셋)
  4. 기본 저장 위치를 선택하세요
  5. 변환 후 원본 삭제 옵션을 켜거나 끄세요
  6. 설정은 자동으로 저장되어 향후 변환에 적용됩니다

📤 내보내기 및 공유

  1. 변환이 완료된 후 미리보기를 탭하세요
  2. 공유 버튼을 탭하세요
  3. 공유 대상을 선택하세요(SNS, 이메일, 메시지 등)
  4. 또는 "Files에 저장"을 사용하여 특정 위치로 내보내기
  5. Mac 또는 다른 기기로의 빠른 전송을 위해 AirDrop 지원

🔧 문제 해결

변환 실패 또는 멈춤

이미지가 손상되었거나 지원되지 않는 형식일 수 있습니다

해결 방법: 다른 이미지를 선택하거나, 먼저 일반적인 형식(예: JPG)으로 변환한 후 다시 시도해 보세요.

Photos 저장 실패

앱이 사진 라이브러리에 대한 쓰기 권한을 가지고 있지 않을 수 있습니다

해결 방법: iOS 설정 > 개인정보 보호 및 보안 > 사진 > Image Converter로 이동하여 권한이 "모든 사진"으로 설정되어 있는지 확인하세요.

이미지를 선택할 수 없음

앱이 사진 라이브러리에 액세스할 수 없을 수 있습니다

해결 방법: iOS 설정 > 개인정보 보호 및 보안 > 사진 > Image Converter로 이동하여 "모든 사진" 또는 "선택한 사진" 권한을 부여하세요.

출력 파일이 너무 큼

품질 설정이 너무 높을 수 있습니다

해결 방법: JPG/WebP의 품질 설정을 낮추세요(권장 70–85%), 또는 더 나은 압축을 위해 WebP/HEIC 형식을 사용하세요.

일괄 변환이 중단됨

기기 저장 공간 부족 또는 앱이 백그라운드로 전송됨

해결 방법: 기기에 충분한 저장 공간이 있는지 확인하고, 변환 중에는 앱을 포그라운드에 유지하거나, 더 작은 배치로 이미지를 처리하세요.

HEIC 변환 실패

Pro 기능 또는 지원되지 않는 iOS 버전

해결 방법: HEIC는 입력 형식으로만 지원됩니다. 출력 형식으로는 사용할 수 없습니다.
