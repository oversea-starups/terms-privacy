❓ Häufig gestellte Fragen

Welche Bildformate unterstützt Image Converter?
Image Converter unterstützt die Konvertierung zwischen folgenden Formaten:
  • Eingabeformate: JPG, JPEG, PNG, HEIC, HEIF, WebP, PDF
  • Ausgabeformate: JPG, PNG, WebP, PDF
Wir erweitern stetig die Unterstützung für neue Formate. Wenn Sie ein bestimmtes Format benötigen, teilen Sie uns dies über eine Funktionsanfrage mit.
Wird die Bildqualität bei der Konvertierung komprimiert?
Sie haben die volle Kontrolle über die Ausgabequalität:
  • JPG-Format: Einstellbare Kompressionsqualität (1–100 %)
  • PNG-Format: Verlustfreie Kompression, bewahrt die Originalqualität
  • WebP-Format: Wahl zwischen verlustbehafteter oder verlustfreier Kompression
  • HEIC-Format: Erhält hohe Qualität bei reduzierter Dateigröße
Die Standardeinstellungen bieten ein gutes Gleichgewicht zwischen Qualität und Dateigröße.
Werden meine Bilder auf einen Server hochgeladen?
Auf keinen Fall. Image Converter verwendet eine vollständig lokale Verarbeitungsarchitektur:
  • Alle Konvertierungen werden auf Ihrem Gerät durchgeführt
  • Für die Nutzung der App ist keine Internetverbindung erforderlich
  • Ihre Bilder verlassen niemals Ihr Gerät
  • Wir folgen einer Datenschutz-first-Designphilosophie
Ihre Daten befinden sich vollständig unter Ihrer Kontrolle.
Werden Stapelkonvertierungen unterstützt?
Ja! Image Converter unterstützt leistungsstarke Stapelkonvertierungen:
  • Mehrere Bilder auf einmal auswählen
  • Dieselben Konvertierungseinstellungen auf alle Bilder anwenden
  • Stapel-Export zu Fotos oder Dateien
  • Ursprüngliche Dateiorganisation beibehalten
Die kostenlose Version hat ein Limit für die Stapelgröße. Upgraden Sie auf Pro für unbegrenzte Stapelkonvertierungen.
Wo werden konvertierte Bilder gespeichert?
Sie können wählen, wo Ihre Bilder gespeichert werden:
  • Fotos: Direkt in der iOS-Fotobibliothek speichern
  • Dateien: In iCloud Drive oder lokale Dateien speichern
  • Teilen: Direkt mit anderen Apps teilen
Sie können in den Einstellungen einen Standardspeicherort festlegen oder bei jeder Konvertierung wählen.
Welche Funktionen sind im Pro-Abonnement enthalten?
Ein Pro-Abonnement schaltet alle Premium-Funktionen frei:
  • Unbegrenzte Stapelkonvertierungen
  • Alle Ausgabeformate
  • Erweiterte Qualitätseinstellungen
  • Benutzerdefinierte Ausgabemaße
  • Werbeentfernung
  • Priorisierter technischer Support
Abonnements werden über den App Store verwaltet und können jederzeit gekündigt werden.
Wie kündige ich mein Abonnement?
Sie können Ihr Abonnement wie folgt kündigen:
  1. Öffnen Sie die Einstellungen-App auf Ihrem iOS-Gerät
  2. Tippen Sie oben auf Ihren Namen
  3. Wählen Sie „Abonnements“
  4. Suchen Sie Image Converter Pro
  5. Tippen Sie auf „Abonnement kündigen“
Nach der Kündigung können Sie die Pro-Funktionen bis zum Ende des aktuellen Abonnementzeitraums weiter nutzen.
Was soll ich tun, wenn die App abstürzt?
Wenn die App abstürzt, versuchen Sie die folgenden Schritte:
  1. Schließen Sie die App vollständig (entfernen Sie sie aus dem App-Switcher) und öffnen Sie sie erneut
  2. Starten Sie Ihr Gerät neu
  3. Suchen Sie nach App-Updates
  4. Suchen Sie nach iOS-Systemupdates
  5. Löschen Sie die App und installieren Sie sie neu (Hinweis: Dies löscht keine gespeicherten Bilder)
Wenn das Problem weiterhin besteht, kontaktieren Sie bitte unser Support-Team mit Ihrem Gerätemodell und iOS-Version.
Welche iOS-Versionen werden unterstützt?
Image Converter unterstützt iOS 15.0 und neuer.

Für die beste Erfahrung empfehlen wir:
  • Die neueste iOS-Version zu verwenden
  • Die App auf dem neuesten Stand zu halten
  • Sicherzustellen, dass Ihr Gerät über ausreichend Speicherplatz verfügt
Wie beantrage ich eine Rückerstattung?
Rückerstattungsanfragen müssen über Apple bearbeitet werden:
  1. Besuchen Sie reportaproblem.apple.com
  2. Melden Sie sich mit Ihrer Apple ID an
  3. Suchen Sie das Image Converter-Abonnement oder den Kauf
  4. Tippen Sie auf „Problem melden“
  5. Wählen Sie einen Rückerstattungsgrund und senden Sie ihn ab
Die Genehmigung von Rückerstattungen liegt im Ermessen von Apple. Bei Fragen können Sie auch direkt den Apple Support kontaktieren.

📖 Benutzerhandbuch

🔄 Schnellstart: Einzelnes Bild konvertieren

  1. Öffnen Sie die Image Converter-App
  2. Tippen Sie auf dem Hauptbildschirm auf die „+“-Schaltfläche oder „Bild auswählen“
  3. Wählen Sie das Bild, das Sie aus Ihrer Fotobibliothek konvertieren möchten
  4. Wählen Sie das Zielformat (JPG, PNG, WebP oder PDF)
  5. Passen Sie Qualitäts- und Größeneinstellungen an (optional)
  6. Tippen Sie auf die Schaltfläche „Konvertieren“
  7. Wählen Sie einen Speicherort (Fotos oder Dateien)

📦 Mehrere Bilder stapelweise konvertieren

  1. Tippen Sie auf die Option „Stapelkonvertierung“
  2. Wählen Sie mehrere Bilder aus (zum Mehrfachauswahl wischen)
  3. Bestätigen Sie Ihre Auswahl und tippen Sie auf „Weiter“
  4. Legen Sie ein einheitliches Ausgabeformat und eine Qualität fest
  5. Tippen Sie auf „Stapelkonvertierung starten“
  6. Warten Sie, bis die Verarbeitung abgeschlossen ist
  7. Speichern oder teilen Sie die konvertierten Bilder stapelweise

⚙️ Benutzerdefinierte Konvertierungseinstellungen

  1. Tippen Sie auf dem Konvertierungsbildschirm auf das Einstellungs-Zahnrad-Symbol
  2. Passen Sie den Ausgabequalitäts-Schieberegler an (nur JPG/WebP)
  3. Legen Sie die Ausgabemaße fest (Original, Benutzerdefiniert oder Voreingestellt)
  4. Wählen Sie den Standardspeicherort
  5. Aktivieren/deaktivieren Sie die Option, das Original nach der Konvertierung zu löschen
  6. Einstellungen werden automatisch gespeichert und auf zukünftige Konvertierungen angewendet

📤 Exportieren & Teilen

  1. Tippen Sie nach Abschluss der Konvertierung auf die Vorschau
  2. Tippen Sie auf die Teilen-Schaltfläche
  3. Wählen Sie ein Ziel zum Teilen (Soziale Medien, E-Mail, Nachrichten usw.)
  4. Oder verwenden Sie „In Dateien speichern“, um an einem bestimmten Ort zu exportieren
  5. Unterstützt AirDrop für schnelle Übertragung auf einen Mac oder andere Geräte

🔧 Fehlerbehebung

Konvertierung schlägt fehl oder friert ein

Das Bild ist möglicherweise beschädigt oder in einem nicht unterstützten Format

Lösung: Versuchen Sie, ein anderes Bild auszuwählen, oder konvertieren Sie das Bild zunächst in ein gängiges Format (z. B. JPG) und versuchen Sie es erneut.

Speichern in Fotos fehlgeschlagen

Die App hat möglicherweise keine Schreibberechtigung für die Fotobibliothek

Lösung: Gehen Sie zu iOS-Einstellungen > Datenschutz & Sicherheit > Fotos > Image Converter und stellen Sie sicher, dass die Berechtigung auf „Alle Fotos“ eingestellt ist.

Bilder können nicht ausgewählt werden

Die App hat möglicherweise keinen Zugriff auf die Fotobibliothek

Lösung: Gehen Sie zu iOS-Einstellungen > Datenschutz & Sicherheit > Fotos > Image Converter und gewähren Sie die Berechtigung „Alle Fotos“ oder „Ausgewählte Fotos“.

Ausgabedatei zu groß

Die Qualitätseinstellung ist möglicherweise zu hoch

Lösung: Verringern Sie die Qualitätseinstellung für JPG/WebP (empfohlen 70–85 %), oder verwenden Sie das WebP/HEIC-Format für eine bessere Kompression.

Stapelkonvertierung unterbrochen

Unzureichender Gerätespeicher oder die App wurde in den Hintergrund geschickt

Lösung: Stellen Sie sicher, dass Ihr Gerät über genügend Speicher verfügt, halten Sie die App während der Konvertierung im Vordergrund, oder verarbeiten Sie die Bilder in kleineren Stapeln.

HEIC-Konvertierung fehlgeschlagen

Pro-Funktion oder nicht unterstützte iOS-Version

Lösung: HEIC wird nur als Eingabeformat unterstützt. Es ist nicht als Ausgabeformat verfügbar.
