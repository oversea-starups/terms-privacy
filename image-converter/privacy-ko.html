3. 사진 라이브러리 권한 사용 안내

📷 사진 라이브러리 읽기 권한

💾 사진 라이브러리 쓰기 권한

변환된 이미지를 사진 라이브러리에 저장하는 데 사용됩니다. 귀하의 확인 후에만 변환 결과가 라이브러리에 기록됩니다.

중요 안내: 모든 이미지 처리는 귀하의 기기에서 로컬로 완료되며, 원본 이미지와 변환된 이미지 모두 귀하의 기기를 벗어나지 않습니다.

6. 귀하의 권리

귀하가 거주하는 지역의 법률 및 규정에 따라, 귀하는 다음과 같은 권리를 가질 수 있습니다:

6.1 GDPR(유럽 연합 사용자)

  • 접근권: 저희가 귀하에 대해 보유하고 있는 데이터를 알 권리
  • 정정권: 부정확한 데이터의 정정을 요구할 권리
  • 삭제권: 귀하의 데이터 삭제를 요구할 권리("잊혀질 권리")
  • 처리 제한권: 데이터 처리를 제한할 권리
  • 데이터 이동권: 일반적인 형식으로 데이터를 얻을 권리
  • 이의 제기권: 특정 유형의 데이터 처리에 이의를 제기할 권리

6.2 CCPA(캘리포니아주 사용자)

  • 저희가 수집하는 개인정보의 카테고리와 구체적인 내용을 알 권리
  • 개인정보 수집 출처의 카테고리를 알 권리
  • 개인정보 수집 목적을 알 권리
  • 개인정보를 공유하는 제3자의 카테고리를 알 권리
  • 수집한 개인정보의 삭제를 요구할 권리
  • 개인정보 보호 권리를 행사했다는 이유로 차별받지 않을 권리

6.3 권리 행사 방법

저희는 극소량의 익명화된 데이터만 수집하므로, 대부분의 사용자 권리 요청은 다음 방법으로 이행할 수 있습니다:
